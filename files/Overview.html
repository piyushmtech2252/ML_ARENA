Iris Flower Classification using KNN

Build a machine learning classification model using the Iris Dataset from scikit-learn and classify flowers into their respective species using the K-Nearest Neighbors algorithm.

15 Marks

Overview

You are required to build a classification model using the Iris Dataset available in scikit-learn. The dataset contains measurements of flowers, and your task is to classify them into their respective species using the K-Nearest Neighbors (KNN) algorithm.
Task 1 — Load and Explore the Dataset

Load the Iris dataset and display the required dataset attributes.

  • Display feature names
  • Display target class labels
  • Use load_iris() from sklearn.datasets
  • Explore attributes like: .data, .target, .feature_names, .target_names
Task 2 — Split the Dataset

Split the dataset into training and testing sets.

  • Use: 80% training and 20% testing
Task 3 — Train the KNN Classifier

Train a K-Nearest Neighbors classification model.

  • Import KNeighborsClassifier from sklearn.neighbors
  • Train the model using .fit()
Task 4 — Predict the Test Data

Predict the flower species for the testing dataset.

  • Use .predict()
  • Generate predictions for test data
Task 5 — Evaluate the Model

Evaluate the performance of the classification model.

  • Calculate Accuracy
  • Generate Confusion Matrix
  • Use model.score() for accuracy
  • Use confusion_matrix()
