Overview

In this examination, you are required to build a binary classification model using the Breast Cancer Dataset available in scikit-learn. The dataset contains features computed from digitised images of breast mass, and your task is to classify tumours as benign or malignant.

All tasks must be implemented using scikit-learn inside a Jupyter Notebook, with each task placed in a separate code cell, clearly labelled and commented.

Tasks to be Performed

1

Load the Dataset

Load the Breast Cancer dataset and explore its key attributes.

  • Display the total number of features in the dataset.
  • Display the target class labels (benign / malignant).
  • Inspect .data, .target, and .feature_names.
from sklearn.datasets import load_breast_cancer
2

Split the Dataset

Divide the data into training and testing subsets.

  • Use an 80 / 20 train-test split.
  • Set a fixed random_state to ensure reproducibility.
3

Train a Logistic Regression Classifier

Build and fit a Logistic Regression model on the training data.

  • Import LogisticRegression from sklearn.linear_model.
  • Train the model on training features and labels.
4

Predict Class Labels

Use the trained model to generate predictions on the test set.

  • Call .predict() on the test features.
  • Store the predicted labels for use in evaluation.
5

Evaluate the Model

Assess classifier performance using two standard metrics.

  • Compute Accuracy.
  • Generate a Confusion Matrix and display its values.
  • Interpret the matrix: True Positives, False Positives, False Negatives, True Negatives.
