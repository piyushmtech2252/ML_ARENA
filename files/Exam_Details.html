Introduction to Computer Vision (AI309E)

Mid Semester Examination (4th Semester)

Overview

In this examination, you are required to perform a series of image processing operations on a given colour image (Lena Image) using Python. The objective of this exam is to evaluate your understanding of basic computer vision concepts, image transformations, and morphological operations.

You must implement all the required tasks using the OpenCV (cv2) library for performing image processing operations and Matplotlib for displaying images in the notebook. Each operation should be implemented through proper Python code, and the output image must be displayed clearly after every task.

Students are required to write their code in a Jupyter Notebook, ensuring that each step is properly organized and clearly labelled.

Tasks to be Performed

You are required to perform the following operations on the given Lena colour image:

1. Read the Image
  • Load the Lena image using an appropriate OpenCV function.
2. Display the Blue Channel of the Image
  • Extract and display only the blue channel from the original colour image.
3. Convert the Image into Grayscale
  • Convert the colour image into grayscale and display the result.
4. Resize the Image
  • Resize the original image to 1000 × 400 pixels.
5. Translate the Image
  • Shift the image by:
  • 200 pixels along the X-axis
  • 100 pixels along the Y-axis
  • Display the translated image.
6. Rotate the Image
  • Rotate the image by 45 degrees around its center.
7. Generate the Negative Image
  • Create the negative of the image by inverting pixel intensity values.
8. Apply Log Transformation
  • Perform logarithmic transformation to enhance lower intensity values.
9. Histogram Equalization
  • Apply histogram equalization to improve image contrast.
10. Morphological Operations
  • Apply the following operations:
  • Erosion
  • Closing
  • Gradient
  • Display results for each operation.

Implementation Guidelines

  • Use OpenCV (cv2) for image processing operations.
  • Use Matplotlib (matplotlib.pyplot) to display images.
  • Each task must be implemented in separate code cells.
  • Clearly label the output of each operation.
  • Add brief comments explaining the steps in your code.
